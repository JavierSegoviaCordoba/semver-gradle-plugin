# Module semver-gradle-plugin

### Apply the plugin

On projects:

```kotlin
// build.gradle.kts
plugins {
    id("com.javiersc.semver") version "$version"
}
```

On settings:

```kotlin
// settings.gradle.kts
plugins {
    id("com.javiersc.semver") version "$version"
}
```
